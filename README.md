# Craftable-Carved-Pumpkin   可合成雕刻的南瓜

**中文** | [**English**](README_en.md)

<br/>
该模组可以让玩家使用剪刀和南瓜合成出雕刻的南瓜，每合成一次剪刀的耐久 -1。
